Baking Pi On C

A guide to make Operating System on Raspberry Pi in C

Downloads

This lesson guides you about how to setup the platform, so that you can also run the codes on Raspberry Pi and experiment out with them freely.

  • Download the packages mentioned below for your Operating System.

For Microsoft Windows

We have used the YAGARTO and MinGW packages. Please visit the YAGARTO website and download and install YGARTO Tools and YGARTO GNU ARM toolchain for Windows. MinGW can be downloaded from here. After installation restart your computer. Note: YAGARTO must be installed to a path with no spaces, e.g. 'C:\YAGARTO\' not 'C:\Program Files\YAGARTO\'.

For Linux

Use the following commands

  • $ wget http://www.cl.cam.ac.uk/freshers/raspberrypi/tutorials/os/downloads/arm-none-eabi.tar.bz2

  • $ tar xjvf arm-none-eabi.tar.bz2

  • $ export PATH=$PATH:$HOME/arm-2008q3/bin


  • Now download the OS template. This template is written by Alex Chadwick who wrote the Cambridge Raspberry Pi tutorials.
